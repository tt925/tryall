# 掼蛋牌局思维训练

一个手机浏览器可用的掼蛋出牌逻辑学习工具，第一版聚焦牌局判断、记牌、推理和进贡还贡练习。

## 使用方法

打开 GitHub Pages 链接即可使用。题目链接可以通过右上角分享按钮复制或发送给朋友。

## 第一版包含

- 出牌逻辑训练
- 记牌优先级训练
- 根据对手出牌推理手牌
- 进贡和还贡判断
- 本机进度统计
- 单题链接分享

## 设计原则

应用按国标/竞技掼蛋的常见规则框架设计题目：两副牌、级牌、红桃级牌逢人配、炸弹、同花顺、四王炸、进贡还贡、抗贡、接风等。产品定位为棋牌知识学习与牌局思维训练，不提供真人对战、随机牌局游戏、虚拟币或赌博相关功能。
